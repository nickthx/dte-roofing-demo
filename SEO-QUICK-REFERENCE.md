## URL Structure

```
/                           → Homepage
/services                   → Services Overview
/services/roof-repair       → Specific Service
/locations/columbus         → Location Landing Page
/blog                       → Blog Archive
/contact                    → Contact Form
/faq                        → FAQ Page
```

**Best Practices**:
- Use hyphens (not underscores)
- Keep URLs short and descriptive
- Include target keyword
- Avoid date parameters
- Use lowercase only

## Content Guidelines

### Keyword Density
- **Target**: 2-3% keyword density
- **Natural Integration**: Keywords should appear naturally
- **Variations**: Use synonyms and related terms

### Content Length
- Homepage: 800-1200 words
- Service Pages: 600-1000 words
- Location Pages: 300-500 words
- Blog Posts: 800-1500 words
- Product/Service Descriptions: 150-300 words

### Readability
- Use short paragraphs (3-4 sentences)
- Include bullet points and lists
- Use subheadings every 300 words
- Write at 8th-grade reading level
- Include images/videos to break up text
